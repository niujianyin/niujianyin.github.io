新浪球通文件资源上传如下:

0.api/config 修改环境变量let env = ‘’
1.npm run build –report
2.把 dist（/Users/njy/workspace/github/2018/04/sina-expert-platform/dist）下文件放到 ballbest（本地svn） (/Users/njy/workspace/svn/web/ballbest/v1/pc)
可以通过publish.sh命令
3.文件上传SVN地址 https://caitong_niujianyin@svn1.intra.sina.com.cn/nostaff_caitong/front_dev/web/ballbest/v1/pc
4.cms（http://cms.pub.sina.com.cn/#file ） 选择新浪球通，选择SVN文件同步(https://caitong_niujianyin@svn1.intra.sina.com.cn/nostaff_caitong/front_dev/web/ballbest/)
上传后文件cms地址举例:http://n.sinaimg.cn/sports/ballbest/v1/pc/js/vendor.js

新浪球通页面部署如下:

0.每次生成的js,css有hash，来防止缓存，页面都的重新发布
1.在match.lottery.sina.com.cn域名下部署html
2.把 dist（/Users/njy/workspace/github/2018/04/sina-expert-platform/dist）下html文件放到 templates (/Users/njy/workspace/svn/deploy/ballbest/templates)
可以通过publish.sh命令
3.文件上传SVN地址 https://caitong_niujianyin@svn1.intra.sina.com.cn/nostaff_caitong/dev/web/match_lottery/app/ballbest/templates
4.登录代码部署系统 https://svnpub.intra.sina.com.cn/index_p.php?project_id=560
选择 app/ballbest/templates 选择文件发布部署
部署后页面访问路径如下:
http://match.lottery.sina.com.cn/ballbest/index/render?__t__=articlePay&pdtIds=expdoc-N1803291415461533096
// 如需配置访问地址转换，找后台配置 如:http://match.lottery.sina.com.cn/ai/account/

校验算法

竞彩足球，亚盘，大小盘：
只选单场单个玩法的情况下，赔率不能低于1.40!
选择最小收益值（每场玩法最低赔率乘积）>理论最小收益(玩法数量乘积)

足彩推荐：
专家推荐胜负14场时，必须推荐14场比赛
专家推荐任选9场时，最多推荐14场中的11场比赛
专家推荐任选9场时，至少推荐9场比赛
选择场数玩法数*2（r9需要算排列组合c9n）>固定金额
